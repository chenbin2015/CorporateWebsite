# 数据初始化脚本

这个脚本用于初始化项目和页面数据。

## 使用方法

### 基本使用

确保后端服务正在运行（默认地址：http://localhost:8080），然后执行：

```bash
npm run init-data
```

### 自定义 API 地址

如果后端运行在不同的地址，可以通过环境变量指定：

```bash
API_BASE=http://localhost:8080/api npm run init-data
```

### 使用认证 Token

如果 API 需要认证，可以通过环境变量提供 token：

```bash
ADMIN_TOKEN=your_token_here npm run init-data
```

## 脚本功能

脚本会创建：

1. **一个示例项目**：名称为"示例项目"
2. **三个页面**，每个页面都包含多个组件：
   - **首页** (`/home`) - 包含 5 个组件（Hero轮播、统计条、信息卡片、内容分割、CTA横幅）
   - **关于我们** (`/about`) - 包含 5 个组件（页面头部、面包屑、图文模块、分区英雄、时间轴）
   - **联系我们** (`/contact`) - 包含 4 个组件（页面头部、联系卡片、表单、合作伙伴）

每个页面都已经填充了示例数据，可以直接在页面搭建器中查看和编辑。
