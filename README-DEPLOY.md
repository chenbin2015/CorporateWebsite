# 🚀 一键部署说明

## 快速开始

在项目根目录执行：

**Windows:**
```bash
deploy.bat
```

**Linux/Mac:**
```bash
chmod +x deploy.sh
./deploy.sh
```

## 文件说明

- `deploy.bat` - Windows 批处理脚本（只能在 Windows 运行）
- `deploy.sh` - Linux/Mac Shell 脚本（只能在 Linux/Mac 运行）

两个文件功能完全相同，只是针对不同操作系统。根据你的系统选择对应的文件执行即可。
